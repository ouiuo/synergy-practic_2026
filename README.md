# Synergy Chat Bot

Учебный чат-бот на языке Python, разработанный в рамках производственной практики
для университета "Синергия". Бот предназначен для ответов на типовые вопросы студентов
по практике и организации учебного процесса на основе распознавания ключевых слов.

## Назначение проекта

Бот используется как простой диалоговый помощник и демонстрирует:
- работу с текстовыми данными;
- базовую обработку естественного языка;
- логику принятия решений на основе ключевых слов;
- модульную структуру Python-приложения.

Проект ориентирован на учебные цели и может быть расширен для реального использования.

## Структура проекта

synergy_chatbot/
├── main.py точка входа, запуск бота
├── config.py конфигурация бота и ответы
├── engine.py логика обработки запросов
├── text_utils.py функции нормализации текста
├── README.md описание проекта


## Используемые конфигурации

Основная настройка бота осуществляется через файл `config.py`.
Все параметры вынесены отдельно от логики, что упрощает изменение поведения бота
без правок основного кода.

### Основные параметры конфигурации

В файле `config.py` используются следующие параметры:

#### BOT_NAME
Имя чат-бота. Используется для идентификации бота и может быть выведено в интерфейсе.

Пример:
```python
BOT_NAME = "SynergyHelper"

HELP_MESSAGE

Сообщение, которое выводится пользователю при запуске бота.
Содержит краткую инструкцию по использованию.

Пример:

HELP_MESSAGE = "Напиши тему одним словом: практика, договор, отчет, ЭИОС, сроки, контакты."

FALLBACK_MESSAGE

Ответ по умолчанию, который возвращается, если бот не смог распознать запрос пользователя.

Пример:

FALLBACK_MESSAGE = "Я не понял запрос. Попробуй задать вопрос иначе."

RULES

Основная конфигурация логики бота. Представляет собой список правил.
Каждое правило содержит:

    список ключевых слов или фраз (patterns);

    текст ответа (answer).

Пример одного правила:

{
    "patterns": ["практика", "производственная практика"],
    "answer": "По практике обычно требуется отчет, индивидуальное задание и аттестационный лист."
}

Настройка и расширение бота
Добавление нового ответа

Чтобы добавить новый сценарий ответа:

    Открой файл config.py

    Добавь новый элемент в список RULES

    Укажи ключевые слова и соответствующий ответ

Пример:

{
    "patterns": ["экзамен", "зачет"],
    "answer": "Информация об экзаменах и зачетах публикуется в ЭИОС."
}

Изменение логики распознавания

Распознавание текста выполняется в файле engine.py.
Перед сравнением текст проходит нормализацию:

    перевод в нижний регистр;

    удаление знаков препинания;

    удаление лишних пробелов.

Функция нормализации реализована в text_utils.py.
При необходимости можно расширить ее, например, добавить:

    лемматизацию;

    удаление стоп-слов;

    поддержку синонимов.

Порог совпадений

В логике бота используется простой механизм оценки совпадений.
Ответ выбирается на основе количества совпавших ключевых слов.
При необходимости можно доработать алгоритм, добавив:

    веса для ключевых слов;

    приоритеты правил;

    более сложные методы обработки текста.

Запуск программы
Интерактивный режим

python main.py

Режим одного запроса

python main.py --once "Какие документы по практике?"
