# Synergy Chat Bot

Простой учебный чат-бот для университета "Синергия".
Бот отвечает на типовые вопросы студентов по практике и учебному процессу
на основе распознавания ключевых слов.

## Требования
Python 3.9+

## Запуск

Интерактивный режим:
python main.py

Один запрос:
python main.py --once "Какие документы по практике?"

## Назначение
Проект разработан в рамках производственной практики
и может использоваться как пример простого диалогового помощника.
